## 🔧 **ÉTAPE 3 : PROCÉDURE INSERTION WORD/DOCX**

### **Configuration Optimale Images :**

#### **1. Paramètres d'Insertion :**
- **Format** : Insertion → Images → À partir d'un fichier
- **Disposition** : Aligné avec le texte
- **Taille** : Largeur 16 cm (ajustement automatique hauteur)
- **Résolution** : Conserver qualité originale

#### **2. Propriétés Images :**
```
Clic droit sur image → Format de l'image → Propriétés :
- Titre : "Figure X: [Titre complet]"
- Description : [Description technique détaillée]
- Balise Alt : "Schéma technique brevet TrustContext Figure X"
```

#### **3. Légendes Formatées :**
```
Format type (sous chaque image) :
**Figure X : [Titre]**
[Description technique détaillée en 2-3 lignes expliquant 
les composants et interactions montrés dans le schéma.]
```

### **Exemple Légende Complète :**
```
**Figure 1 : Architecture Générale Système TrustContext**

Schéma technique illustrant l'organisation modulaire du système de contexte 
de confiance multi-niveaux avec six composants principaux intégrés : 
classificateur de criticité 4 niveaux, mécanisme d'escalade automatique, 
générateur de tokens cryptographiques sécurisés, système de validation 
multi-couches, gestion superviseurs autorisés et audit trail immutable 
avec métriques de performance temps réel.
```
